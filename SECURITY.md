# Security Policy

## Supported Versions
We release patches for security vulnerabilities in the following versions:

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability
If you discover a security vulnerability, please send an email to contact@example.com with details.
We will review and respond as quickly as possible.

Please do not publicly disclose the vulnerability until it has been addressed.
